# 项目简介

## 核心功能
- 数据爬取
- 数据管理
- 数据可视化
- 就业推荐

## 适用场景
- 企业招聘部门
- 人力资源公司
- 求职者个人

## 技术特点
- 基于 Django 和 MySQL 的框架构建
- 使用 HTML 进行前端展示
- 集成协调过滤推荐算法，优化职位推荐效果
